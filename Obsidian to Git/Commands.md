## Step 1 :- 

Go to the vault that you want to push to github or gitlab.

## Step 2 :- 

```
git init
```

## Step 3 :-
```
git add .
```

## Step 4 :-
```
git commit -m "Initial commit of Obsidian vault"
```

## Step 5 :-
```
git branch
```

## Step 6 :-
```
git remote add origin https://github.com/shashank-baikadi/SQL_Notes.git
```

## Step 7 :-

```
git push -u origin main
```
